I've customized and utilized the Software Developer Master Portfolio, originally developed by Ashutosh Hathidara and various contributors, for my personal portfolio. This portfolio template is built on the React.js library, offering a clean, beautiful, responsive, and fully customizable showcase for a software developer's work, skills, projects, and more.

To make this portfolio my own, I followed the detailed customization instructions provided, including updating personal and GitHub information, selecting a theme, and deploying the site. The original project is licensed under the MIT License, ensuring open-source accessibility and modification.

Technologies used in this portfolio include React, GraphQL, Apollo Boost, and more, with illustrations provided by UnDraw. My customization process involved personalizing sections such as About Me, Skills, Projects, Experience, and Contact Information, among others, to accurately reflect my professional profile and achievements.

This portfolio stands as a testament to the collaborative spirit of the open-source community, and I am grateful for the foundation provided by Ashutosh Hathidara and all the contributors to the masterPortfolio project.
